How to Create Detail Pages for All Courses

This guide explains how to create detail pages for each course in the Learnly programming notes website. Each detail page will include a PDF preview, detailed course information, and download option.

Step-by-Step Implementation

Step 1: Create a New HTML File

Create a new HTML file named after the course (e.g., python-detail.html, javascript-detail.html).

Use the detail-template.html as your starting point.

Step 2: Update Metadata and Content

Replace the placeholder content with course-specific information:

  • Update page title and meta tags
  • Replace COURSE_NAME with the actual course name
  • Update course category, page count, and other metadata
  • Write a course-specific description
  • List course features and learning outcomes

Step 3: Update Image and PDF Links

Make sure to update the following elements for each course:

  • Course thumbnail image path (src="pdf-images/CourseName.png")
  • PDF preview source (src="pdf/CourseNameNotesForProfessionals.pdf.pdf")
  • Download link with the correct file path and name

Step 4: Update Main Index Links

Modify index.html to link to your new detail page:

<a href="coursename-detail.html" class="course-card-image-wrap w-inline-block">
  <img loading="lazy" src="pdf-images/CourseNameGrow.png" alt="Course Name" class="course-thumbnail">
  <div class="card-badge"><div class="text-xs">Category</div></div>
</a>

Example Content Template

Here's a template for the course description section:

<div class="course-description">
  <h2>About this Course</h2>
  <p>[Brief introduction to the language/technology]</p>
  
  <p>This comprehensive guide covers everything from [Language] basics to advanced topics. You'll learn about [key concepts], and more. The notes also cover popular libraries like [library names] and frameworks like [framework names].</p>
  
  <p>Whether you're a beginner looking to learn your first programming language or an experienced developer wanting to add [Language] to your skill set, these notes provide a thorough reference for [Language] development.</p>
</div>

<div class="course-features">
  <h2>What You'll Learn</h2>
  <ul>
    <li>[Feature 1]</li>
    <li>[Feature 2]</li>
    <li>[Feature 3]</li>
    <li>[Feature 4]</li>
    <li>[Feature 5]</li>
  </ul>
</div>

Final Checklist

  • Page title and meta tags updated
  • Course name updated throughout the page
  • Course description is relevant and informative
  • Learning outcomes are specific to the course
  • Image paths are correct
  • PDF file path is correct
  • Download link is working
  • Back button links to index.html
  • Main index.html links to the detail page
